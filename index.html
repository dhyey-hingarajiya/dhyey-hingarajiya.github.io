// Design Overview

Implemented a full 8-bit ripple carry adder through the complete ASIC flow. The design was specified in synthesizable Verilog RTL, verified with directed and random testbenches in Synopsys VCS, synthesized to a 45nm standard cell library using Design Compiler, and then taken through place & route in ICC2.

// Timing & Constraints

Applied 800MHz clock constraint with 30% uncertainty margin. PrimeTime STA confirmed zero setup/hold violations post-route. Critical path runs through the carry chain of the LSB full adder to MSB. Area and power tradeoffs explored by comparing flat vs. hierarchical synthesis strategies.

module fa ( input A, B, Cin, output Sum, Cout ); assign Sum = A ^ B ^ Cin; assign Cout = (A & B) | (B & Cin) | (A & Cin); endmodule // Synthesis report excerpt // Total cell area: 124.13 (normalized) // Combinational area: 124.13 // Critical path: 1.21 ns // Power: 58.3 µW (dynamic + leakage)

// Signal Pipeline

Seismic signals were loaded from raw sensor CSV data. Preprocessing included DC offset removal, normalization, and segment windowing. A 4th-order Butterworth bandpass filter (0.1–10 Hz) was designed to isolate the P-wave frequency band from broadband noise.

// Detection Algorithm

Short-Term Average / Long-Term Average (STA/LTA) detector used to identify P-wave onset. FFT-based spectral analysis confirmed event signatures. Results compared against USGS ground truth catalog to validate detection accuracy across varying magnitude events.

% Butterworth bandpass: 0.1 – 10 Hz fs = 50; [b, a] = butter(4, [0.1 10] / (fs/2), 'bandpass'); x_filt = filtfilt(b, a, x_raw); % STA/LTA detector sta_len = 5; lta_len = 30; threshold = 3.5; ratio = stalta(x_filt, sta_len, lta_len); events = ratio > threshold;

// System Model

Quadrotor modeled as a 6-DOF rigid body in Simulink. Attitude dynamics linearized around hover equilibrium. Separate PID controllers designed for roll, pitch, and yaw axes. Plant transfer function derived from physical parameters (mass, inertia, thrust coefficient).

// PID Tuning

Gains tuned using Ziegler-Nichols method as initial estimate, then refined via Simulink's Response Optimizer to minimize ITAE criterion. Bode and Nyquist plots confirmed gain margin >12 dB and phase margin >60° for robust stability. Step response validated against design specs.

// System Architecture

MPU-6050 IMU (3-axis accelerometer + gyroscope) interfaced via I²C to a microcontroller. Sensor fusion using complementary filter combines accelerometer and gyroscope data to compute accurate tilt angle. Raw acceleration magnitude computed from all three axes for threshold comparison.

// Detection Algorithm

Fall characterized by three phases: free-fall (low g-force), impact (high g-force spike), and post-fall posture (tilt angle deviation). Algorithm gates on all three conditions sequentially to minimize false positives from normal activities like sitting or jumping.

// Compute acceleration magnitude float acc_mag = sqrt(ax*ax + ay*ay + az*az); if (acc_mag < FREEFALL_THRESHOLD) { fall_state = FREEFALL; } else if (fall_state == FREEFALL && acc_mag > IMPACT_THRESHOLD) { fall_state = IMPACT; start_posture_timer(); } else if (fall_state == IMPACT && tilt_angle > TILT_THRESHOLD) { trigger_alert(); // FALL CONFIRMED }
